## Como publicar no GitHub Pages

1. Criar repositório com o nome exato: `sunda290.github.io`
2. Fazer upload de todos os arquivos desta pasta
3. Ir em **Settings → Pages → Source: main branch / root**
4. Aguardar ~2 minutos → site disponível em `https://sunda290.github.io`
